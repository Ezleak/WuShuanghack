# 搭建环境
1. 克隆此代码仓库
2. 在IDEA中运行

在根目录控制台输入:
```
./gradlew setupDevWorkspace idea genIntellijRuns build
```
3. 打开IDEA.
4. 打开此项目目录
5. 点击导入Gradle项目
